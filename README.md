# AI Career Platform

An AI-powered career development platform that helps users improve their resumes, practice interviews, and plan skill roadmaps.

## Features

Resume Analyzer
- Upload resume (PDF)
- AI scoring and feedback
- Keyword gap analysis

AI Interview Simulator
- Dynamic interview questions
- AI evaluation of answers
- Hiring decision simulation

Career Roadmap Generator
- Interactive skill mindmap
- Role-based learning paths

Mock Interview Booking
- Schedule interview sessions
- Automated email meeting link

Admin Dashboard
- View bookings
- Delete bookings
- Manage sessions

## Tech Stack

Backend
- FastAPI
- Python

AI
- OpenAI API
- Sentence Transformers
- Scikit-learn

Frontend
- Jinja2
- HTML / CSS / JavaScript
- D3.js (mindmap)

Infrastructure
- SQLite
